This program is a simple jumping game.
Made by Maxime Bisaillon (Maxy Arthes).
May 22, 2017.
Version : 1.0.0
Python : 3.6.0

You need to have python 3.6.0 or higher and pygame installed to run this game.
Python : (https://www.python.org/)
Pygame : https://www.pygame.org


Control :
Space = Jump
Left arrow key = Move the character to the left.
Right arrow key = Move the character to the right.

Level :
Level 1 = 0 points
Level 2 = 15 points
Level 3 = 51 points
Level 4 = 151 points
Level 5 = 251 points
